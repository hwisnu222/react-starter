## Installation

clone repo

```
$ git clone git@github.com:hwisnu222/react-starter.git
```

install dependencies

```
$ yarn install
```

start project

```
$ yarn start
```

or if you use npm

```
$ npm start
```

open browser and access this url [http://localhost:3000](http://localhost:3000)
